**Employee and Department Data Analysis**  
Course: DS PGC Course 2 — Assignment 2

## Project overview
This repository contains my solution for **Assignment 2** of DS PGC Course 2.  
The task is an Excel-based analysis of Employees and Departments datasets to help improve HR processes and departmental efficiency. Tasks include conditional formatting, data validation, Power Query transformations, Power Pivot relationships and pivot tables, Excel Tables, macros, named ranges, and per-department comparisons.

## Sheet list (what to expect inside the workbook)
Each of the 10 solutions is on a separate sheet and the sheet names match the task numbers:

- `Task-1_ConditionalFormatting` — Highlight salaries > 15,000 and hires within last 4 years.
- `Task-2_DataValidation` — Department ID validation using Departments dataset.
- `Task-3_PowerQuery` — Combine first & last name; extract username from email.
- `Task-4_PowerPivotRelationship` — Relationship between Employees and Departments via Department ID.
- `Task-5_PivotSalaryByDept` — Power Pivot table: total & average salary per department.
- `Task-6_Table_Employees` — Converted Employees dataset to Excel Table named `Employees Table`.
- `Task-7_Filter_Sales` — Filter showing only employees from Sales department.
- `Task-8_Macro_AddEmployee` — Button + recorded macro to add a new employee.
- `Task-9_NamedRange_AvgSalary` — Named range and formula to compute average salary.
- `Task-10_DeptSalaryComparison` — Conditional formatting to highlight highest salary per department.

## How to open and review
1. Download `DataScienceCourse2Assignment 2Answer.xlsx`.  
2. Open using **Microsoft Excel (desktop)** to ensure Power Query, Power Pivot, macros, and Data Model features work correctly.  
3. To inspect macros: go to **Developer → Macros** (or press `Alt+F8`) and view the recorded macro assigned to the button on the `Task-8_Macro_AddEmployee` sheet.  
4. For Power Pivot & Data Model: open the **Power Pivot** window (if available) to view the relationship created between tables.
